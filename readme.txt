Create an unlimited number of sidebars and assign unlimited number of widgets.

== Description ==

This plugin requires the [Pluton](https://plutonwp.com/) theme to be installed.

== Installation ==

1. Upload `pluton-custom-sidebar` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Done!

== Frequently Asked Questions ==

= I installed the plugin but the custom sidebar do not appear =

This plugin will only work with the [Pluton](https://plutonwp.com/) theme.
